A Terraform-native approach to AWS Landing Zone / Control Tower

A "Terraform Landing Zone" or "Terraform Control Tower" is a tool that uses Terraform to:
  1. Quickly create new AWS accounts
  2. Configure AWS accounts with a standard security baseline
  3. Define a best-practices multi-account setup

Gruntwork Landing Zone helps you achieve all these goals using Terraform, allowing you to create and manage AWS accounts as code.
