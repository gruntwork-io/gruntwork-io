== EKS Migration Guides

The migration process for the EKS cluster, core services, and applications from Reference Architecture v1 (uses
infrastructure-modules) to v2 (uses https://github.com/gruntwork-io/terraform-aws-service-catalog[Service Catalog]) is
slightly different from migrating other modules because the ordering matters and because some post-migration steps are
required.
